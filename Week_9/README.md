# IT Helpdesk Ticket Registration System

## Purpose of the Application
The IT Helpdesk Ticket Registration System is a simple modular Python application designed for City University Malaysia. It allows students to report technical issues (e.g., login errors, Wi-Fi issues, printer errors) and automatically assigns a technician based on the issue's priority level.

## Tech Stack
* **Language:** Python 3
* **Tools:** Git & GitHub
